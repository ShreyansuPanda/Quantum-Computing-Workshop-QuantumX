## Overview

This repository contains a collection of **interactive quantum computing experiments** built using Qiskit.  
These notebooks are designed for beginners to **learn, visualize, and experiment** with core quantum concepts.

---

## Topics Covered

- Quantum Random Number Generation  
- Superposition and Bloch Sphere Visualization  
- Entanglement  
- Deutsch Algorithm  
- Grover’s Search Algorithm  
- Grover Iterations & Probability Oscillation  
- Quantum Teleportation  

---

## Features

- Interactive sliders and controls  
- Visualizations (Bloch sphere, histograms, circuits)  
- Hands-on experimentation with parameters  
- Beginner-friendly implementations  

---

## How to Use

1. Open the Colab link above  
2. Run all cells  
3. Use sliders and inputs to explore different behaviors  
4. Modify the code to experiment further
